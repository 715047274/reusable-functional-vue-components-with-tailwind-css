# Reusable Functional Vue.js Components with Tailwind CSS

[![Patreon](https://img.shields.io/badge/patreon-donate-blue.svg)](https://www.patreon.com/maoberlehner)
[![Donate](https://img.shields.io/badge/Donate-PayPal-blue.svg)](https://paypal.me/maoberlehner)

This is an example project for the following article: [Reusable Functional Vue.js Components with Tailwind CSS](https://markus.oberlehner.net/blog/reusable-functional-vue-components-with-tailwind-css/)

## Build Setup

```bash
# Install dependencies.
npm install

# Serve with hot reload.
npm run serve

# Build for production with minification.
npm run build
```
